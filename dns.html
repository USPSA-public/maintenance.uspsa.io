Our website has been moved to a new server. If you're experiencing issues accessing the site, your DNS cache may need to be refreshed.

How to Refresh Your DNS Cache

Windows

1. Press Win + R to open the Run dialog.
2. Type cmd and press Enter to open the Command Prompt.
3. In the Command Prompt, type the following command and press Enter:
   ipconfig /flushdns
4. You should see a confirmation message that the DNS Resolver Cache has been successfully flushed.

macOS

1. Open Terminal (you can find it in Applications > Utilities).
2. Depending on your macOS version, type one of the following commands and press Enter:
   macOS Monterey or later:
   sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
   
   macOS Big Sur and earlier:
   sudo killall -HUP mDNSResponder
3. Enter your password when prompted.
4. There will be no confirmation message, but the DNS cache will be cleared.

Linux

1. Open a terminal window.
2. Type the following command and press Enter:
   sudo systemctl restart network-manager
   Or, depending on your distribution:
   sudo service network-manager restart
3. The DNS cache will be cleared.

iOS (iPhone/iPad)

1. Go to Settings > Wi-Fi.
2. Tap the i next to your connected network.
3. Tap Forget This Network.
4. Reconnect to the same Wi-Fi network.

Android

1. Go to Settings > Connections > Wi-Fi.
2. Long press on your connected Wi-Fi network and select Forget Network.
3. Reconnect to the same Wi-Fi network.
